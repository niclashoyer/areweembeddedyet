HAL Implementation

Implementations of embedded-hal for microcontroller families and systems running some OS.

NOTE You may be able to find even more HAL implementation crates by searching for the embedded-hal-impl and embedded-hal keywords on crates.io!

OS

  • bitbang-hal

    Implements embedded-hal traits by bitbanging

  • linux-embedded-hal

    Implementation of the `embedded-hal` traits for Linux devices
